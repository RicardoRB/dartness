The core libraries of the Dartness framework.

See documentation guides at https://ricardorb.github.io/dartness/#/.

Classes

Bind
Annotation for a bindable property.
Body
Annotation indicating a method parameter should be bound to the body of the http request.
Controller
Annotation to identify a class as a controller. A controller is a class that can be used to handle requests.
Dartness
A server that delivers content, such as web pages, using the HTTP protocol by HttpServer.
DartnessInterceptor
DartnessInterceptor is an interface in order to handle the request before it is executed and handle the response after the request is executed.
DartnessMiddleware
DartnessInterceptor is an interface in order to handle the request before it is executed this can be helpful if you want to log the request or do something else before the request is executed.
DartnessPipeline
A helper that makes it easy to compose a set of DartnessMiddleware, DartnessMiddleware and a Handler.
DartnessRouter
A router that can be used to handle requests.
DartnessRouterHandler
A router handler for handling request for a ClassMirror with Controller with his metadata and the method MethodMirror with the metadata.
DartnessServer
An interface that defines the methods that a server must implement.
Delete
Route annotation for delete request.
Get
Route annotation. Routes Get requests to the specified path.
Head
Route annotation. Routes Head requests to the specified path.
Header
Annotation indication to specify a custom response header
HttpCode
An annotation to indicate what HTTP status code must be returned.
LogRequestsInterceptor
Middleware which prints the time of the request, the elapsed time for the inner handlers, the response's status code and the request URI.
Options
Route annotation. Routes Options requests to the specified path.
Patch
Route annotation. Routes Patch requests to the specified path.
PathParam
Annotation to bind a property to a path parameter.
Post
Route annotation. Routes Post requests to the specified path.
Put
Route annotation. Routes Put requests to the specified path.
QueryParam
Annotation to bind a property to a query parameter.
